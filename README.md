# Join - Kanban Project Management Tool

Join ist eine interaktive Kanban-Board-Anwendung, die im Rahmen der Ausbildung zum Frontend-Entwickler erstellt wurde. Sie hilft Teams dabei, Aufgaben effizient zu organisieren, zu priorisieren und den Fortschritt zu verfolgen.

## ✨ Features
* **User Management:** Registrierung und Login für verschiedene Teammitglieder.
* **Kanban Board:** Aufgaben per Drag & Drop zwischen den Status (To Do, In Progress, Await Feedback, Done) verschieben.
* **Task Management:** Erstellen, Bearbeiten und Löschen von Aufgaben inklusive Kategorien und Prioritäten.
* **Contacts:** Ein integriertes Adressbuch zum Zuweisen von Aufgaben an Teammitglieder.
* **Summary:** Eine Dashboard-Übersicht über alle aktuellen Aufgaben und Deadlines.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
* **Storage:** Firebase Realtime Database / LocalStorage integration
* **Version Control:** Git & GitHub

## 📂 Projektstruktur
Das Projekt folgt einer klaren Struktur, um die Wartbarkeit zu gewährleisten:
* `/js` - Enthält die gesamte Logik, unterteilt in Module (Board, Contacts, Tasks).
* `/CSS` - Alle Stylesheets inklusive Schriftarten.
* `/assets` - Vorlagen (Templates) und Icons.
* `index.html` - Der Haupteinstiegspunkt der Anwendung.
